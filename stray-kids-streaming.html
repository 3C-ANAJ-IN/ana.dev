stayflix

O projeto tem como objetivo central oferecer um espaço para consumo do grupo de kpop stray kids, descobrirem e interagirem com conteúdos relacionados ao gênero. O site funciona como uma vitrine dos principais projetos do grupo musical, além de reunir informações sobre a formação do grupo e de suas redes sociais.

Esse projeto é especial para mim, pois foi minha primeira vez trabalhando com html, além de despertar em mim a curiosidade por programação. Além disso, para esse projeto desenvolvi uma animação no método stop motion, desenhando frame a frame para o cabeçalho e identidade visual do site.
